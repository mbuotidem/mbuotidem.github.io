About Me

Background
I am a fullstack software engineer with a background in product management and development. As such, nothing gives me more pleasure than shipping great products that serve customer's needs. I'm most productive in C# and naturally, I'm pretty conversant with the Microsoft stack - .NET Core, Entity Framework, Microsoft SQL Server and Microsoft Azure. However, I've also dabbled in both Ruby and Python and I am currently exploring the Node.js ecosystem.
Philosophy
I enjoy learning new technology and languages. For example, although my day-to-day work is in Angular, I am currently learning React. I've always found that such playful exploration and continuous learning not only keeps my skills sharp but also is genuinely gratifying.
Personal Interests
On weekends, I can be found either showing off my salsa dance moves or making hard tackles on the soccer pitch.

Projects


PeerTutor

PeerTutor is a web application where users can book tutoring sessions from a pool of college peers. Users can select tutors based on ratings, book and pay for sessions, and leave reviews. It is written using C# and the Microsoft .NET Core Framework.
interro.us

interro is a responsive web application where users can create and play their own quizzes. Users can also play quizzes created by other users in a public quiz area. It is written using Angular for the frontend and C# and .NET Core for the backend api.

nocontact.us

NoContactBot is a mobile responsive chatbot that provides support to those with broken hearts. It is written in C# using the Microsoft Bot Framework and Azure Cognitive Services.
