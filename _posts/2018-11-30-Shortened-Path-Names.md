**Shortened Path Names**

If the path contains spaces, use the shortened path name. For example, `C:\Progra~1\Java\jdk1.8.0_65`

For Windows users on 64-bit systems
Progra~1 = 'Program Files'
Progra~2 = 'Program Files(x86)'
